FAQ

Does AI Insight read the full paper?

No. AI Insight uses PubMed abstract + metadata only (no full text). If details are not present in the abstract, the insight should indicate they are not reported.

Can I edit the PubMed query directly?

Yes. Advanced Search supports editing the underlying query, adding clauses, and using common PubMed tags (e.g., [tiab], [pt], [jour], [au]) and date ranges.

Is this a general medical search tool?

CardioThoraxia is tailored to cardiothoracic surgery workflows with a curated taxonomy organized by clinical practice area.

Where can I get help or suggest improvements?

Use the Support link in the footer. The app also supports submitting topic suggestions to improve the catalog over time.
